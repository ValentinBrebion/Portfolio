Cybersécurité

La cybersécurité est un aspect important dans notre filière puisqu'il est le problème majeure dans les entreprises. Attaque au physhing, injection SQL. En tant que développeur, nous devons constamment nous former pour contrer ces menaces et nous permettre d'assurer à l'entreprise ainsi qu'aux clients une constante intégrité des données et une confidentialité optimale des données.

Phishing

L’hameçonnage ou phishing est une technique utilisée par des fraudeurs pour obtenir des renseignements personnels dans le but de perpétrer une usurpation d'identité.

Injection SQL

L'injection SQL est un groupe de méthodes d'exploitation de faille de sécurité d'une application interagissant avec une base de données.

Ransomwares

Un rançongiciel, logiciel rançonneur, logiciel de rançon ou logiciel d'extorsion, est un logiciel malveillant qui prend en otage des données personnelles.

DDOS

Une attaque par déni de service est une attaque informatique ayant pour but de rendre indisponible un service, d'empêcher les utilisateurs légitimes d'un service de l'utiliser.
